<h1 align="center">Nix Config</h1>

## Install Nix

```bash
curl --proto '=https' --tlsv1.2 -sSf -L https://install.determinate.systems/nix | \
  sh -s -- install
```

## apply changes

```bash
./scripts/apply.sh
```
